# PersonalExpenseManager


### Project Overview

This project is a personal expense management application developed using C# and Windows Forms.

The main goal of this project is to practice object-oriented programming concepts, user interface design, data management, and file handling.

### Features

- User Login
- Add New Expenses
- Expense Categories
- Expense Date Registration
- DataGridView Display
- Total Expense Calculation
- Maximum Expense Detection
- Budget Usage Percentage
- Category Filtering
- Expense Deletion
- CSV Export
- CSV Import
- Live System Clock
- User Information in StatusStrip

### Technologies

- C#
- .NET 8
- Windows Forms
- CSV File Handling

### Learning Objectives

- Object-Oriented Programming (OOP)
- Working with Lists
- Event Handling
- DataGridView Usage
- File Operations
- Windows Forms Development
