# PersonalExpenseManager
A personal expense management application developed using C# and Windows Forms.

Features
User Login
Add New Expenses
Expense Categories
DataGridView Display
Total Expense Calculation
Maximum Expense Detection
Budget Usage Percentage
Category Filtering
Export to CSV
Import from CSV
Live Clock and User Status Bar
Technologies
C#
.NET 8
Windows Forms
CSV File Handling
Notes

This project was developed as a learning project to practice C# programming, Windows Forms development, file handling, and basic data management concepts.
